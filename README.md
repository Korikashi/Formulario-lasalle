# Formulario de Contacto - Universidad La Salle

## Descripción
Proyecto web que permite registrar mensajes de contacto en una base de datos.

## Tecnologías
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Funcionalidades
- Registro de datos desde formulario
- Validación de campos
- Visualización de mensajes
- Prueba de conexión (test.php)
